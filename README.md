# Cursor Course

Next.js app with TypeScript and Tailwind CSS. Includes Overview, Research Assistant, Research Reports, API Playground (dashboards), and Invoices.

## Setup

See [SETUP-INSTRUCTIONS.md](./SETUP-INSTRUCTIONS.md) for install and run steps.  
For Supabase (API Playground), see [README-SUPABASE.md](./README-SUPABASE.md).

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run start` — start production server
- `npm run lint` — run ESLint
